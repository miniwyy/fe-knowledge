快捷键
===

> 全局操作

- Command + p  // 快速打开
- Command + w  // 关闭窗口

> 基本操作

- Option + 向上键  // 向上移动一行代码
- Option + 向下键  // 向下移动一行代码
- Option + shift + 向上键  // 向上复制一行代码
- Option + shift + 向下键  // 向下复制一行代码
- Command + 回车键  // 向下插入一行
- Command + Shift + 回车键  // 向上插入一行
- Command + 向上键  // 跳到文件首部
- Command + 向下键  // 跳到文件尾部
- Ctrl + 数字键  // 跳到对应数字的标签页

> 选择

- Command + I  // 选择光标所在的一行
- option + 鼠标点击  // 插入多个光标

> 导航

- Ctrl + G + 数字  // 跳到某行
- Command + P  // 跳到某文件

> 显示

- Command + `+`  // 放大
- Command + `-`  // 缩小
- Command + b  // 显示、隐藏侧边栏
- Ctrl + Shift + g  // 显示Git面板
- Command + Shift + x  // 显示插件面板
- Ctrl + `  // 显示终端
