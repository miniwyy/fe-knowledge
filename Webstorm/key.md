快捷键
===

```dash
⌘——Command

⌃ ——Control

⌥——alt

⇧——Shift

⇪——Caps Lock

fn——功能键就是fn
```

  - fn + 方向键   // 进行翻页
  
  - Control + ~  // 替换主题
  
  - alt + /  // 代码补全
  
  - Command + L  // 查找某行
  
  - Shift + Shift  // 查找文件
  
  - Command + Shift + f  // 查找内容
  
  - Command + Shift + Delete // 光标上一个位置
  
  - fn + Command + 向左鍵/向右键  // 光标跳到文件顶部/尾部
