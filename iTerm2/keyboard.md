[快捷键](https://yugasun.com/post/iterm2-shortcut-key.html)
===

- Command + /  // 显示光标位置，挺酷炫的
- Ctrl + a 【fn + 向左方向键】  // 命令文本开头
- Ctrl + e 【fn + 向右方向键】  // 命令文本结尾
- Ctrl + r  // 搜索命令历史
- Command + Enter // 满屏显示
- open . // 在当前目录下打开finder
- Command + f // 所查找的内容会被自动复制
- Command + d // 横着分屏
- Command + shift + d // 竖着分屏
- Command + r 只是换到新一屏，不会像 clear 一样创建一个空屏
- ctrl + u 清除当前行
- ctrl + w 删除光标之前的单词
- ctrl + k 删除到文本末尾
- Command + 方向键 // 切换tab
- ctrl + y // 粘贴剪切板的内容
