纯函数
===

- 纯函数这样是一种函数，相同的输入，永远会得到相同的输出，而且没有任何可观察的副作用

> `Erlang` 语言的作者 `Joe Armstrong` 说，面向对象语言的问题是，它们永远都要随身携带那些隐式的环境。你只需要一个香蕉，但却得到一个拿着香蕉的大猩猩...以及整个丛林

- 例子：

```javascript
var a = [1, 2, 3, 4, 5];

// 纯的方式
a.slice(0, 3);
// => [1, 2, 3]

a.slice(0, 3);
// => [1, 2, 3]

a.slice(0, 3);
// => [1, 2, 3]

// 不纯的方式
a.splice(0, 3);
// => [1, 2 ,3]

a.splice(0, 3);
// => [4, 5]

a.splice(0, 3);
// => []
```
